<p align="center">
  Pelicano keeps text, images, and file copies in one searchable history, then brings them back fast with preview, tray access, and <code>Ctrl+Shift+V</code>.
</p>

## Snapshot

<table>
  <tr>
    <td width="33%" valign="top">
      <strong>Capture</strong>
      <br />

    </td>
    <td width="33%" valign="top">
      <strong>Recall</strong>
      <br />
