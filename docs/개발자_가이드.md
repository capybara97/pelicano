# Spaste 개발자 가이드

## 개요
Spaste는 Windows 10/11 환경을 대상으로 한 기업용 클립보드 매니저다. 텍스트 복사 시 `Plain Text Only` 강제 적용, Markdown 변환 복사, 이미지/이미지 파일 히스토리 저장을 기본 기능으로 제공한다.

## 프로젝트 구조
- `Spaste/Program.cs`: STA 진입점과 애플리케이션 컨텍스트 실행
- `Spaste/AppContext.cs`: 트레이, 클립보드 감시, 히스토리 저장, 설정 반영
- `Spaste/MainForm.cs`: 히스토리 목록, 개별 복사 버튼, 미리보기 UI
- `Spaste/SettingsForm.cs`: 운영 설정 변경 UI
- `Spaste/ClipboardListener.cs`: `WM_CLIPBOARDUPDATE` 기반 감시
- `Spaste/ClipboardService.cs`: 재시도 포함 클립보드 읽기/쓰기
- `Spaste/TextStripper.cs`: 텍스트 정규화, Markdown 변환, 해시 계산
- `Spaste/HistoryManager.cs`: LiteDB 저장소
- `Spaste/TrayManager.cs`: NotifyIcon + 컨텍스트 메뉴
- `Installer/Spaste.iss`: Inno Setup 설치 스크립트
- `scripts/Convert-PngToIco.ps1`: PNG를 ICO로 변환하는 내부 스크립트

## 빌드 전제 조건
- .NET 8 SDK
- NuGet 네트워크 접근 가능 환경
- Inno Setup 6.x (`ISCC.exe`)

## 배포 빌드
`build.bat`는 다음 단계를 자동으로 처리한다.
1. `Spaste\Resources\app.png`를 `app.ico`로 변환
2. NuGet 복원
3. `win-x64`, self-contained, single-file publish
4. Inno Setup 설치 파일 생성

## 핵심 동작 흐름
1. `ClipboardListener`가 클립보드 변경 메시지를 받는다.
2. `AppContext`가 텍스트/이미지/파일 드롭인지 판별한다.
3. 텍스트면 `TextStripper`가 일반 텍스트 정규화와 Markdown 변환을 수행한다.
4. 이미지면 `%APPDATA%\Spaste\images`에 PNG로 저장한다.
5. 메모리 히스토리와 LiteDB를 동기화한 뒤 `MainForm`을 갱신한다.

## 메모리/성능 메모
- 히스토리는 기본 200건으로 제한한다.
- DB는 전체 교체 전략을 사용하지만 최대 데이터량이 작아 운영 부담이 낮다.
- 이미지 미리보기는 선택 시에만 로드하고, 이전 이미지는 즉시 Dispose 한다.
