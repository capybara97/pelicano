# Pelicano 개발자 가이드

## 개요
Pelicano는 Windows 10/11용 WinUI 3 기반 클립보드 히스토리 앱이다. 텍스트, 이미지, 파일 드롭을 로컬에 저장하고, `Ctrl+Shift+V` 전역 단축키와 시스템 트레이를 통해 빠르게 다시 호출할 수 있다.

## 현재 구조
- `Winui/App.xaml.cs`: WinUI 앱 진입점, 창 생성, 트레이 초기화, 종료 흐름
- `Winui/MainWindow.xaml.cs`: 메인 Fluent UI 셸, 검색/미리보기/설정/업데이트 UI
- `Winui/PelicanoHost.cs`: 클립보드 감시, 히스토리 저장, 설정 반영, 업데이트 요청
- `Winui/WindowMessageMonitor.cs`: `WM_CLIPBOARDUPDATE`, `WM_HOTKEY` 수신
- `Winui/TrayManager.cs`: Win32 `Shell_NotifyIcon` 기반 시스템 트레이
- `Winui/ClipboardService.cs`: 텍스트, 이미지, 파일 드롭 읽기/쓰기
- `Winui/HistoryManager.cs`: LiteDB 저장소
- `Winui/SettingsDialog.xaml.cs`: 설정 저장, 자동업데이트 확인 UI
- `Winui/Services/UpdateService.cs`: Vercel 매니페스트 조회, 설치 파일 다운로드, SHA-256 검증
- `Installer/Pelicano.iss`: Inno Setup 설치 스크립트
- `build.bat`: restore, publish, installer 생성 자동화

이 저장소의 제품 실행 경로는 WinUI `Pelicano.exe` 하나만 사용한다. 레거시 WinForms 앱은 제품 경로에 포함되지 않는다.

## 빌드 전제 조건
- .NET 8 SDK
- NuGet 네트워크 접근 가능 환경
- Inno Setup 6.x (`ISCC.exe`)

## 로컬 빌드
```powershell
dotnet restore .\Pelicano.sln --configfile .\NuGet.Config
dotnet build .\Pelicano.sln
```

## 배포 빌드
`build.bat`는 다음 단계를 자동으로 수행한다.
1. `Winui\Resources\app.png`를 `app.ico`로 동기화한다.
2. `Pelicano.sln` 복원
3. `Winui\Pelicano.WinUI.csproj`를 `win-x64`, self-contained, single-file로 publish
4. `Installer\Pelicano.iss`로 다국어 설치 파일 생성

설치기는 다음을 포함한다.
- 한국어/영어 언어 선택 UI
- 언어별 이용약관/개인정보 처리방침 페이지
- `{app}\docs` 아래 개발/보안 문서
- `{app}\docs\legal` 아래 `license-ko.txt`, `privacy-ko.txt`, `license-en.txt`, `privacy-en.txt`

## 런타임 데이터 경로
- 설정: `%APPDATA%\Pelicano\settings.json`
- 히스토리 DB: `%APPDATA%\Pelicano\history.db`
- 이미지 캐시: `%APPDATA%\Pelicano\images`
- 로그: `%APPDATA%\Pelicano\logs`
- 업데이트 다운로드: `%LOCALAPPDATA%\Pelicano\updates`

## 핵심 동작 흐름
1. `WindowMessageMonitor`가 클립보드 변경과 전역 단축키 메시지를 받는다.
2. `PelicanoHost`가 텍스트/이미지/파일 드롭을 판별한다.
3. 텍스트는 정규화 후 필요 시 plain text only로 다시 적용한다.
4. 이미지는 `%APPDATA%\Pelicano\images` 아래에 PNG로 저장한다.
5. 항목을 LiteDB에 반영하고 `MainWindow` 목록과 미리보기를 갱신한다.

## 자동업데이트 매니페스트
업데이트 주소는 설정창 입력이 아니라 앱 내부에 하드코딩되어 있다.

- 매니페스트: `https://qfqxifaqnympxjbz.public.blob.vercel-storage.com/version.json`
- 기본 설치 파일: `https://qfqxifaqnympxjbz.public.blob.vercel-storage.com/Pelicano-Installer.exe`

앱은 위 `version.json`을 읽고, 새 버전이 있으면 진행 바를 띄운 뒤 설치 파일을 바로 내려받아 업데이트를 시작한다. 종료 직전 `%APPDATA%\Pelicano\history.db`와 `settings.json`을 다시 저장하므로 히스토리는 유지된다.

`version.json`은 다음 JSON 형식을 읽는다.

```json
{
  "version": "0.3",
  "installerUrl": "https://qfqxifaqnympxjbz.public.blob.vercel-storage.com/Pelicano-Installer.exe",
  "sha256": "optional_sha256_hex",
  "releaseNotes": "optional notes"
}
```

`installerUrl`을 생략하면 앱은 하드코딩된 기본 설치 파일 URL을 사용한다.

## 확인 포인트
- WinUI 기본 실행 파일: `Pelicano.exe`
- Light/Dark, Mica/Acrylic, 시스템 액센트 컬러 자동 반영
- 닫기 시 트레이 백그라운드 유지
- `Ctrl+Shift+V` 전역 단축키 복원
- 설치기 언어 선택에 따라 약관/개인정보 페이지와 태스크 라벨이 현지화됨
