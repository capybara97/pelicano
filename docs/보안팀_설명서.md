# Pelicano 보안팀 설명서

## 제품 목적
Pelicano는 기업 내부 PC에서 텍스트, 이미지, 파일 드롭 복사 이력을 사용자 로컬 영역에만 저장하는 WinUI 3 기반 클립보드 히스토리 앱이다.

## 보안 설계 원칙
- 기본 저장 위치는 사용자별 `%APPDATA%\Pelicano`
- 관리자 권한 없이 실행, 설정 변경, 자동 시작 등록 가능
- 런타임 텔레메트리 없음
- 주요 이벤트는 로컬 감사 로그에 기록

## 저장 위치
- 설정: `%APPDATA%\Pelicano\settings.json`
- 히스토리 DB: `%APPDATA%\Pelicano\history.db`
- 이미지 캐시: `%APPDATA%\Pelicano\images`
- 로그: `%APPDATA%\Pelicano\logs`
- 업데이트 설치 파일: `%APPDATA%\Pelicano\updates`

## 기록되는 이벤트
- 애플리케이션 시작/종료
- 텍스트, 이미지, 파일 드롭 캡처
- 사용자 재복사
- 설정 변경
- 오류 발생 시 예외 정보

## 자동 시작
자동 시작은 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`의 `Pelicano` 값으로만 관리된다. 시스템 전체 `HKLM` 등록은 사용하지 않는다.

## 자동업데이트
- 앱에 하드코딩된 HTTPS `version.json` 주소 기준으로 동작
- 설치 파일은 `%APPDATA%\Pelicano\updates`에 저장
- `sha256`이 제공되면 다운로드 파일 무결성 검증 수행
- 설치는 현재 앱 종료 후 별도 설치 파일 실행 방식
- 업데이트 직전 현재 설정과 히스토리 DB를 다시 저장

## 설치기 법률 문서
- Inno Setup은 한국어/영어 언어 선택을 지원
- 각 언어별 이용약관/개인정보 처리방침 문서를 설치 UI에 표시
- 설치 후 `{app}\docs\legal`에 한국어/영어 법률 문서 4종을 함께 배치

## 제거 정책
- Inno Setup uninstall 지원
- 설치 제거 시 프로그램 파일 삭제
- 설치 제거 시 Pelicano 사용자 데이터 폴더만 정리하도록 설정되어 있음

## 운영 시 확인 포인트
- 보안 솔루션이 `AddClipboardFormatListener`와 `RegisterHotKey`를 차단하지 않는지 확인
- 전역 단축키 `Ctrl+Shift+V` 충돌 여부 확인
- 대용량 이미지 사용 시 `%APPDATA%\Pelicano\images` 용량 모니터링
